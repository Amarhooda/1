This is about HTML

What is HTML?


- Stands for HyperText Markup Language. Defines the structure and content of web pages. Not a programming language, but a markup language using tags to tell browsers how to display content.

Basic Structure:

An HTML document starts with <html> and ends with </html>.
The content you see goes inside the <body> tag.
Headings use <h1> for most important, <h2> for less important, and so on.
Paragraphs are defined with <p>.
Links are created with <a> tags and their href attribute specifies the target URL.
Images are embedded with <img> tags and their src attribute specifies the image source.
